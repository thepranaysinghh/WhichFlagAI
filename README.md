# WhichFlagAI 🚩

>  AI-powered Relationship & Marriage Decision Support System

## 🌐 Live Demo
[whichflagai.vercel.app](https://whichflagai.vercel.app)

## 💡 What is WhichFlagAI?
WhichFlagAI is a specialized AI that helps users identify Red, Yellow & Green flags in their relationships and rishtas — giving structured clarity before major life decisions.

## ✨ Features
- 🚩 Red / Yellow / Green Flag Analysis
- 🧠 Psychological Insight
- 🧭 Final Recommendation (Proceed / Observe / Reconsider)
- 📊 Clarity Score
- ⚠️ Risk Alert System
- 💬 Multi-language Support (Hindi, English, Hinglish)

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- AI: Groq API (LLaMA 3.3 70B)
- Deployment: Vercel
- CI/CD: GitHub Actions

## 🚀 Deployment
Auto-deployed via GitHub Actions CI/CD pipeline on every push to main branch.
